📖 คู่มือการใช้งานระบบ

อธิบายการทำงานและขั้นตอนการใช้งาน

1. การกรอกและแก้ไขข้อมูล (สำคัญ)

  • การพิมพ์ภาษาไทย: ระบบนี้ใช้ตารางแบบพิเศษ คุณต้อง ดับเบิลคลิก (Double-Click) ให้เคอร์เซอร์กระพริบในช่องก่อน ถึงจะพิมพ์ภาษาไทยได้โดยที่สระไม่ลอย
  • การลบแถวที่พิมพ์ผิด: ห้ามลบข้อความทิ้งให้เป็นช่องว่างเฉยๆ ให้กดไอคอน 🗑️ (ถังขยะ) ที่คอลัมน์ขวาสุด เพื่อนำข้อมูลและรูปภาพออกจากระบบอย่างสมบูรณ์
  • การเพิ่มทีละเยอะๆ: หากต้องการเพิ่มอุปกรณ์ 50 ชิ้น แนะนำให้กดปุ่ม "ดาวน์โหลด (Excel)" เพื่อโหลดไฟล์ไปทำในโปรแกรม Excel ก่อน แล้วค่อยกด "นำเข้าไฟล์" ทีหลัง
  • จุดระวัง: ทุกครั้งที่มีการแก้ไข/ลบ/นำเข้า ต้องกดปุ่ม "💾 บันทึก" เสมอ ข้อมูลถึงจะส่งไปเก็บที่เซิร์ฟเวอร์

2. ระบบแจ้งชำรุดและอัปโหลดรูปภาพ

เมื่อมีอุปกรณ์เสีย แอดมินสามารถแนบรูปถ่ายความเสียหายเข้าระบบได้ทันที เพื่อให้ช่างประเมินอาการ:

  1. เปลี่ยนคอลัมน์ "สถานะ" เป็น "ชำรุด"
  2. กดไอคอน 📸 (กล้อง) ที่คอลัมน์จัดการขวาสุด
  3. เลือกรูปภาพในมือถือหรือคอมพิวเตอร์ (ขนาดไม่เกิน 5MB)
  4. ระบบจะสร้างปุ่ม 🖼️ รูป ไว้ในช่องหมายเหตุให้อัตโนมัติ (อย่าลืมกดบันทึก)
  5. การลบรูป: สามารถกดที่ปุ่ม 🖼️ รูป เพื่อเปิดรูปขึ้นมา แล้วกดปุ่ม "ลบรูปภาพนี้" ได้เลย

3. ระบบ QR Code ประจำเครื่อง

  • สามารถกดพิมพ์ป้าย QR Code ไปแปะติดอุปกรณ์ได้ โดยกดปุ่ม "🖨️ พิมพ์ QR" เพื่อพิมพ์รวดเดียวทั้งสถานี หรือกดไอคอนรูป QR ท้ายแถวเพื่อพิมพ์ทีละเครื่อง
  • เมื่อนำมือถือมาสแกน QR Code ระบบจะพาเข้าสู่หน้าเว็บ พร้อมโชว์ ประวัติการซ่อมและการเปลี่ยนสถานะ ของเครื่องนั้นตั้งแต่เริ่มใช้งานให้ดูทันที

4. การส่งรายงานประจำเดือน

  • เมื่อตรวจสอบข้อมูลครบถ้วนแล้ว ให้กดปุ่ม "ส่งรายงาน" ระบบจะสรุปยอดทั้งหมดให้ดูอีกครั้งเพื่อยืนยัน
  • สถานีที่ส่งแล้ว จะมีป้ายกำกับสีเขียวว่า ✅ ส่งแล้ว แสดงที่หน้าหลัก

1. การเปิดดูและตรวจสอบตาราง

ในหน้านี้ ข้อมูลตารางทั้งหมดจะถูก ล็อกสิทธิ์ให้อ่านได้อย่างเดียว (Read-Only) เพื่อความปลอดภัยของข้อมูลและป้องกันการเผลอกดลบ

  • สามารถตรวจสอบรายการอุปกรณ์ สถานะ และรูปแบบการเชื่อมต่อของหน่วยท่านได้
  • หากต้องการดูรูปภาพความเสียหายของอุปกรณ์ ให้คลิกที่ปุ่ม 🖼️ รูป ในคอลัมน์ "หมายเหตุ"
  • กดไอคอนคิวอาร์โค้ดที่คอลัมน์ขวาสุด เพื่อดูรหัส QR ประจำเครื่องนั้นๆ

2. การ Export ไฟล์เพื่อนำไปแก้ไข

กรณีที่อุปกรณ์ในสถานีมีการเปลี่ยนแปลง เพิ่ม/ลด หรือชำรุด ให้ดำเนินการดังนี้:

  1. คลิกที่ปุ่ม "📥 ดาวน์โหลด (Excel)" ที่มุมบนขวาของหน้าต่าง
  2. ระบบจะทำการเซฟข้อมูลตารางทั้งหมดลงเครื่องคอมพิวเตอร์ของคุณเป็นไฟล์ Excel (.xlsx)
  3. เปิดไฟล์ Excel นั้นด้วยโปรแกรม Microsoft Excel ปกติ เพื่อทำการพิมพ์แก้ไขข้อมูล, เติมอุปกรณ์ใหม่ หรือเปลี่ยนสถานะให้เรียบร้อย
  4. เซฟไฟล์ Excel ที่แก้ไขเสร็จแล้วให้เรียบร้อย โดยตั้งชื่อไฟล์เป็น สน.ส.วจ.___ หรือ ชุด ส.วจ.___ หรือ ผ__.(ในที่ตั้ง) และตามด้วยวันที่แก้ไข เช่น 17-05-2569

3. การส่งไฟล์ให้ส่วนกลาง (โฟลเดอร์ ผกม.)

  • นำไฟล์ Excel ที่คุณแก้ไขข้อมูลเสร็จเรียบร้อยแล้ว ไปวางลงใน "โฟลเดอร์ รับ-ส่ง ข้อมูล ผกม. สำหรับ สถานี/ชุด (ส่งผ่าน Paperless สำหรับ ในที่ตั้ง)"
  • จากนั้นแจ้งให้แอดมินส่วนกลาง ผกม. (ภายใน:2298, DDN:9401) เพื่อรับทราบ
  • แอดมินส่วนกลางจะนำไฟล์ Excel ของท่านไป Import เข้าระบบใหญ่ เพื่อให้ข้อมูลอัปเดตตรงกันต่อไป
